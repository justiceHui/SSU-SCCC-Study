# 2022 SCCC 봄 초급 강의

### 개요

* 스터디 내용
  * PS(Problem Solving)에서 사용하는 기초적인 개념과 실제 문제에 적용하는 방법
* 권장 대상
  * PS를 처음 접해보는 사람
  * 문제를 풀어본 경험이 적은 사람
  * 알고리즘 공부를 했지만 기초적인 내용을 잘 응용하지 못하는 사람
* 선수 조건
  * C/C++ 문법
* 시간 및 장소
  * 5월 첫째 주 ~ 6월 첫째 주 수요일/토요일 저녁 7시
  * Zoom으로 진행
  * 1시간 * 10회, 총 10시간
